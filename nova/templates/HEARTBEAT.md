# Heartbeat

Use this file to tell Nova what it should periodically check when idle.

Guidelines:
- Keep the instructions concrete.
- If there is nothing to do, say so clearly.
- If there are recurring checks, explain what counts as actionable.

Example:
- Check whether there are unfinished tasks in `.tasks/`.
- If there are blocked tasks that now have enough context to continue, resume them.
- If nothing needs action, skip.
